Exercise UI (Instructor Mode)

The EUI allows you, as the instructor, to record interactions to serve as the basis for a solution, which the student will later be assessed against. Note the EUI in instructor mode has a red bar at the top.

  1. The EUI is launched via the CAT (previous step).
  2. Load the M4 model by selecting the M4 Carbine button in the upper-right corner
  3. Click on the relevant areas of the object to initiate actions. A context menu shows the available actions
  4. Perform the exercise steps in sequence. Click 'M4 Solution Example' above for the correct order.
  5. If you make a mistake during the exercise, select the Reset button to start over.
  6. Select the "Save Solution" button to write the exercise file to the repository.

Result: the exercise file is written to the exercise repository.

Next, you will refine the solution in the ESE.
