Content Assembly Tool (CAT)

The CAT allows you to build a 3D scenario for an interactive training exercise. You can load the semantically annotated models (from the previous step) and place them in the scene.

  1. Start the tool by clicking the 'Launch CAT' button below the image. The empty scene will start out gray.
  2. Select and load the example background asset 'Shooting Range'.
  3. Open the 'Shooting Range' menu on the left. Ensure that 'visible' and 'auto load' are both checked.
  4. Select and load the example asset: M4 Carbine.
  5. Open the 'M4 Carbine' menu on the left. Ensure that 'visible' and 'behaviors' (not shown) are both checked, but that 'auto load' is not.
  6. Move the M4 one-half unit high by entering the number 0.5 in the Z field under the M4 Carbine menu.
  7. Orient the M4: Check rotZ. Enter 90 in the rotation. Uncheck rotZ. Repeat for rotY with -90 rotation.
  8. Move the camera higher by entering the number 0.75 in the Z field under the Camera menu.
  9. Enter a path and exercise name in the 'Path' field. Then select Save Exercise at the bottom.
  10. Click the Open button to work on this scene in the Exercise UI.

Next, you will specify the exercise solution in the Exercise UI.
