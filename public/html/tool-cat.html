Content Assembly Tool (CAT)

The CAT allows you to build a 3D scenario for an interactive training exercise. You load the semantically annotated models and place them in the scene.

  1. Start the tool by clicking the 'Launch CAT' button to the right.
  2. Select and load the example background asset 'Shooting Range'.
  3. Open the 'Shooting Range' menu on the left. Ensure that 'visible' and 'auto load' are checked.
  4. Select and load the example asset: M4 Carbine.
  5. [CHECK FOR MISSING STEPS, e.g. camera.]
  6. Move the M4 up by entering the number 2 in the Z field under the M4 Carbine menu.
  7. Rotate the M4 around one axis at a time to place it in the proper orientation.
  8. Enter the path and exercise name in the 'Path' field. Then select Save Exercise.

Your initial exercise file, containing this scene information, is written to the server and can then be opened in the Exercise UI next.
