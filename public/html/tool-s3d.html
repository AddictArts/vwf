Semantic 3D Annotation Tool (S3D)

The S3D annotation tool enables a 3D content author to associate semantic classes with corresponding nodes of a 3D model.

  1. Start the tool by clicking the 'Launch S3D' button to the right.
  2. Choose a Flora ontology and a 3D model from the predefined files.
  3. Link an ontology class to a 3D node by selecting and clicking 'Link'
  4. To remove links, click 'Delete'
  5. Examine properties by right-clicking on class and selecting 'Info'
  6. Save annotation (.s3d format)

Your .s3d file, containing this link information, is written to the server and becomes available for use by the Content Assembly Tool.
